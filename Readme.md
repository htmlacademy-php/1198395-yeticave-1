# Установка приложения

- на основе файла `config.sample.php` создать файл `config.php` и вписать свои настройки конфигурации
- накатить структуру БД, выполнив sql запросы из файла `./bd/schema.sql`. В файле `./bd/queries.sql` хранятся
тестовые запросы к БД.
